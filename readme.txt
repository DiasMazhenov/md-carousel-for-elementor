Professional nested carousel widget for Elementor.

== Description ==
Each slide in this carousel is a full flex-container, allowing you to drag and drop any Elementor widgets inside.

== Installation ==
1. Upload the plugin folder to the /wp-content/plugins/ directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
